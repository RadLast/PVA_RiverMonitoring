Přihlašování uživatelů
--------------------------------------------------------------
- používá tabulku AspNetUsers
	UserName: jméno k přihlášení
	FullName: celé jméno uživatele
	AccessLevel: role (Read, Write, Admin)
- role:
	Read: Může si data z databáze pouze přečíst
	Write: Data může přepisovat, mazat, přidávat stanice
	Admin: Může vše - má správu uživatelů
